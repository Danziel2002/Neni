Functia cosinus

`f : RR rarr RR`

`f(x) = cos(x)`

`2pi - perioada`

PROPRIETATI

`f. bijectivă: f : [0,pi]rarr [-1,1]`

`f. surjectivă: f : RR rarr [-1, 1]`

`f. "injectivă": f : [0, pi] rarr RR`

`-1 <= cos(x) <= 1; AA x in RR`

Funcția cosinus este strict descrescatoare pe intervalul `[0, pi]` si strict crescatoare pe intervalul `[pi, 2pi]`

Funcția Cosinus este funcție para => `cos(-x) = cos(x); AA x in RR`

Funcția nu este injectivă pe `RR` dar restricția la `[0. pi]` este injectivă

Funcția este surjectivă pe `RR`

Funcția cos este periodică (T0 = `2pi`) => `cos(x) = cos(x+2pi) = cos(x+4pi) = ... = cos(x+ 2Kpi); K in ZZ`
