Functia sinus

`f : RR rarr RR`

`f(x) = sin(x)`

`2pi - perioada`

PROPRIETATI

!OBS: Funcția este strict crescătoare pe `[-pi/2, pi/2]`

Graficul functiei sinus este o curbă numita sinusoidă

`f. bijectivă: f : [-pi/2, pi/2]rarr [-1,1]` - inversabilă

`f. surjectivă: f : RR rarr [-1, 1]`

`f. "injectivă": f : [-pi/2, pi/2] rarr RR`

Funcția Sinus este funcție impară `=> sin(-x) = -sin(x); AA x in RR`

Funcția Sinus este strict crescătoare pe intervalul `[0, pi/2] si [(3pi)/2, 2pi]` si strict descrescătoare pe intervalul `[pi/2, (3pi)/2]`

Funcția nu este injectivă pe `RR` dar restricția la `[-pi/2; pi/2]` este injectivă

Funcția este surjectivă pe `RR`

FuncȒia nu este bijectivă pe `RR` dar este bijectivă pe restricția `[-pi/2; pi/2] rarr [-1;1]` Inversa funcției find arcsin

Funcția Sin este periodică(T0 = `2pi`) => `sin(x) = sin(x+2pi) = sin(x+4pi) = ... = sin(x+ 2Kpi); k in ZZ`
