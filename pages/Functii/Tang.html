`f(x) = tg(x)`

`E = RR - {(2K + 1) pi/2 | K in ZZ}`

`E = ... U ((-3pi)/2, -pi/2) U (-pi/2;pi/2) U (pi/2,(3pi)/2) U ..`

`pi - perioada`

`tg: R "/" {pi/2 + Kpi | K in ZZ} rarr RR; tg(x) = sin(x)/cos(x)`

`tg: R "/" {pi/2 + Kpi | K in ZZ} rarr RR; tg(x) = sin(x)/cos(x)`

Funcția tangentă este strict crescatoare pe intervalul `(-pi/2, pi/2)`

Avem a,b `in (-pi/2, pi/2), a < b` => `tg(a) < tg(b)`

!OBS: Datorita periodicității, funcția tangentă este strict crescătoare pe `(-pi/2 + Kpi, pi/2 + Kpi); AA K in ZZ`

Funcția tangentă este funcție impară => `tg(-x) = -tg(x); AA x in RR`

Funcția tg este periodica (T0 = `2pi`) => `tg(x) = tg(x+2pi) = tg(x+4pi) = ... = tg(x+ Kpi); K in ZZ`

Funcția nu este injectivă pe `RR \ {pi/2 + Kpi; K in ZZ}` si este injectivă pe `(-pi/2; pi/2)`

Funcția este surjectivă pe `RR \ {pi/2 + Kpi; K in ZZ}`

Funcția nu este bijectivă pe `RR \ {pi/2 + Kpi; K in ZZ}` si este bijectivă pe `f : (-pi/2;pi/2) rarr RR`

Funcția este inversabilă = `f^-1 : RR rarr (-pi/2; pi/2) ; f^-1 (x) = arctg(x)`

Funcția cosinus este strict descrescatoare pe intervalul `[0, pi]` si strict crescatoare pe intervalul `[pi, 2pi]`
