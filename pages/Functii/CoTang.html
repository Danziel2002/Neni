Functia cotangenta

`f : E rarr RR`

`f(x) = ctg(x)`

`E = RR - {Kpi | K in ZZ}`

`E = RR / {kpi, k in ZZ}`

`pi - perioada`

`ctg: R "/" {Kpi | K in ZZ} rarr RR; ctg(x) = cos(x)/sin(x)`

PROPRIETATI

Funcția cotangentă este strict descrescătoare pe intervalul `(0, pi)`

Datorită periodicitati , funcția cotangentă este strict dscrescătoare pe `(-Kpi, Kpi) AA K in ZZ`

`E = 'RR / {kpi, k in ZZ}

`pi - perioada`

`ctg: R "/" {Kpi | K in ZZ} rarr RR; ctg(x) = cos(x)/sin(x)`

Funcția cotangentă este funcție impară => `ctg(-x) = -ctg(x); AA x in RR "/" {Kpi | K in ZZ}`

Funcția ctg este periodică (T0 = `pi`) => `ctg(x) = ctg(x+pi) = ctg(x+2pi) = ... = tg(x+ Kpi); K in ZZ`

Funcția nu este injectivă pe `RR "/" {Kpi, K in ZZ}`, dar este pe (o, `pi`)

Funcția este surjectivă pe `RR "/" {Kpi, K in ZZ}`

Funcția nu este bijectivă pe `RR "/"{Kpi, K in ZZ}`

Funcția este descrescatoare si bijectivă pe `f: (0, pi) rarr RR`

Funcția este inversibilă: `f^-1 : RR rarr (0, pi); f^-1 (x) = arcctg(x)
