Functia arcsinus

`f. "este bijectivă pe" [-1,1]`

`f. "este surjectivă pe" [-1,1]`

`f. "este injectivă pe" [-1,1] `

Funcția ArcSin : `[-1;1] rarr [-pi/2; pi/2]` verifica egalitatile:

Inversa funcției `f: f^-1 :[-1;1] rarr [-pi/2; pi/2]` se numeste arcsinus si se noteaza arcsin

Funcția ArcSin : `[-1;1] rarr [-pi/2; pi/2]` verifica egalitatile:

`sin(arcsin x) = x; AA x in [-1,1]`

`arcsin(sin x) = x; AA x in [-pi/2; pi/2]`

Graficul funcției arcsin este simetrciul graficului funcției `f` in raport cu dreapta `y = x`

Funcția `f` Bijectiva => `AA a in [-1;1]`, ecuația sin(x) = a are soluția pentru intervalul [`-pi/2; pi/2`], anume X = arcsin(a)

funcția arcsinus este strict crescătoare

`arcsin(sin x) = x, AA x in [-pi/2, pi/2], iar sin(arcsin x) = x, AA x in [-1,1]`

arcsin(-x) = -arcSin(x); `AA x in [-1, 1]` => `f`. arcinsinus este impară
